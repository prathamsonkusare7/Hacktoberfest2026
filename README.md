# ProPlumb Plumbing Website Landing Page

Welcome to the ProPlumb Plumbing Website!

## Overview
The ProPlumb website serves as an online platform where users can learn about our plumbing services, schedule appointments, and get in touch with our team.

## Features
- **Service Descriptions:** Detailed information on various plumbing services offered.
- **Appointment Scheduling:** Easy-to-use forms to book plumbing services.
- **Contact Information:** Get in touch with ProPlumb for inquiries or emergencies.

## Technologies Used
- HTML, CSS, JavaScript for front-end development
- Node.js for back-end services

## Getting Started
1. Clone the repository
2. Open `index.html` in a web browser to view the landing page.
